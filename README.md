## Cara Menjalankan Aplikasi (XAMPP)
1. Pastikan XAMPP sudah terinstall dan Apache aktif.
2. Letakkan folder proyek di `C:\xampp\htdocs\POKEMON` (sudah sesuai contoh).
3. Buka browser, akses: `http://localhost/POKEMON/`
4. Jika perlu akses file JSON lewat PHP, pastikan file/folder memiliki permission yang benar.
5. Untuk memeriksa error PHP, lihat `C:\xampp\apache\logs\error.log` atau tampilkan error di browser (dev only).
